This directory stores repository-owned workspace defaults and project-level runtime configuration.

Included here:
- AGENTS.md / HEARTBEAT.md / SOUL.md / TOOLS.md / USER.md
- cron/jobs.json
- custom_output_skills.json

Not copied here:
- live sessions
- memory summaries and graph data
- uploaded media
- runtime process files

Those remain under /root/.nanobot/workspace or /root/.nanobot/media at runtime.
