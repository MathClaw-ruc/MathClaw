## 3. 优先复习知识点
- **函数定义域与值域**：重点复习分母不为零、被开方数非负、真数大于零的限制条件。
- **含参函数分类讨论**：重点复习参数分界点确定、导数零点求解及区间符号判断。
- **导数应用（单调性/极值/最值）**：重点复习求导公式、单调区间求解及闭区间最值比较。
- **函数概念基础**：回顾函数三要素，强化图像变换与性质辨析。

## 4. 重点纠错方向
- **定义域忽略**：每道题草稿纸顶部必须先写“定义域为..."，否则视为无效练习。
- **分类讨论遗漏**：建立“临界点清单”，解方程后必须验算所有根，参数为零情况必讨论。
- **端点值检验缺失**：闭区间最值问题建立“三步检查法”（极值点、左端点、右端点）。
- **解题步骤跳跃**：对照高考评分标准，关键变形单独成行，禁止直接写结论。

## 5. 推荐练习量

- **导数计算**：5 道纯计算题（确保求导准确无误）。
